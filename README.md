# CS155 Final Project: Pokémon Quiz
## Author: Matthew Nolan

## Project Description:

For my final project, I decided to do a Pokémon quiz game. Last semester in CS150, one of our programming assignments was to create a quiz game. For that assigment, I also chose to do a Pokémon quiz, because it is one of my favorite subjects, and pretty simple to make a quiz about. I thought it would be interesting to see my progression from last semester. Now, writing a quiz game in C++ seems like such and easy and simple task, and so writing a very similar program in assembly, a way more challenging language for me, would show how I've grown as a programmer and understand more complicated concepts and code.

The quiz has 10 questions. Each question has 3 options: A, B, and C. The user is able to input uppercase or lowercase letters for their answer and recieve the correct points. At the end, the program displays the users score, a message correlated to what score they got (bad, okay, good), and the option to try again and restart the quiz.

## How to run the program:

The program is run on LC3 Tools v2.0.2

To run the program, open the file quiz.asm in the LC3 simulator. Select the breakpoint at line x3009, and run the program. No additional changes to the registers or memory is necessary.
